This repository contains the artifact accompanying our ASE 2026 paper submission. The following is the description of the required contents -

- `Analyzer/` – This folder conatins the java source code for extracting repositories, generating results
- `coding/` – Completed coding of each annotator
- `data/` – Groundtruth dataset and generated results
- `scripts/` – scripts for generating graphs
- `Survey Forms/` – All human survey forms along with survey responses
